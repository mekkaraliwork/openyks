# openyks — GitHub Pages

Statik landing page. Build adımı yok.

## Yayınlama

1. Bu klasördeki dosyaları (`index.html`, `onboarding.html`, `.nojekyll`) repo'nun **ana dizinine** koy.
2. GitHub → Settings → Pages → Source: `main` branch, `/ (root)`.
3. Birkaç dakika içinde `https://<kullanıcı>.github.io/<repo>/` adresinde yayında.

## Dosyalar

- `index.html` — Landing (A+ kopya: Hero V4, TrustBar, Features, How it works, Neden openyks karşılaştırması, FAQ, Final CTA)
- `onboarding.html` — Placeholder akış
- `.nojekyll` — Jekyll'i devre dışı bırakır
